# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.1.3] - 2026-04-20

### Added

- Added author, homepage, bugs, and engines fields to package.json for npm best practices.

## [0.1.2] - 2026-04-20

### Fixed

- Fixed GitHub Pages deployment workflow to correctly build and deploy all slide versions

## [0.1.1] - 2026-04-20

### Fixed

- Fixed oversized prologue section in built presentations

### Added

- GitHub Pages site: https://c-harding.github.io/slidev-addon-timing-bar/

## [0.1.0] - 2026-04-20

### Added

- Initial public release of `slidev-addon-timing-bar`.
- Section-based timing bar for Slidev presenter view.
- Per-section durations and buffer points.
- Live progress tracking and color-coded progress arrow.
- End-time mode and duration mode toggle.
- Timing review modal with actual vs. planned times.
- AI agent skill for automated section configuration.

### Changed

- Version bump to 0.1.0 for first stable release.
