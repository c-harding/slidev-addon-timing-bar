## [0.1.0] - 2026-04-20

### Added

- Initial public release of `slidev-addon-timing-bar`.
- Section-based timing bar for Slidev presenter view.
- Per-section durations and buffer points.
- Live progress tracking and color-coded progress arrow.
- End-time mode and duration mode toggle.
- Timing review modal with actual vs. planned times.
- AI agent skill for automated section configuration.

### Changed

- Version bump to 0.1.0 for first stable release.
